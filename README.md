# WebHair

WebHair is a modern web project for AI-powered hair styling.  
It includes both mobile and desktop layouts, fully responsive and designed for a smooth user experience.

## Project Info

**Project URL**: https://github.com/tharukakaru/hairappweb

## How to Run Locally

Make sure you have Node.js installed.

```sh
# Step 1: Clone the repository
git clone https://github.com/tharukakaru/hairappweb.git

# Step 2: Navigate into the project folder
cd webhair

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
